Reservation management

Reservations

As a reserver, you can make reservations for resources. As a manager, you can create, update, and delete reservations.

Reservation form

Resource, start time, and end time are required. Fill in the details below. Use Create for new items, Update for existing ones, and Delete to remove.

Choose the resource you want to reserve. Click the field to see available options.

start
end

Select a valid start and end time for the reservation. The end time must be after the start time.

You have the option to add a note to your reservation. This can be used to provide additional information or special requests related to your reservation. Please keep the note concise and relevant to your reservation.

Active status

When enabled, the reservation is active.

Reservation list

Click an item to view, edit or delete it.
